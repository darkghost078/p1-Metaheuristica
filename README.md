# Instalacion

## Entorno virtual

En la terminal ejecuta:

```bash
python -m venv .venv
```

Para activarlo ponemos:

```bash
source .venv/bin/activate
```

Para instalar las dependencias:

```bash
pip install -r requirements.txt
```
